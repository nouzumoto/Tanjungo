Telegram Translations Helper

OFF

Available hotkeys:

  • Ctrl+Enter or y → Apply 1st suggestion & move to next
  • Ctrl+Enter (in field) → Submit suggestion & move down
  • Ctrl+A or i → Add a new translation
  • Ctrl+E or c → Edit the first suggestion
  • Ctrl+K → Quick search selected text
  • Ctrl+Shift+X → Cut & Wrap Link
  • Ctrl+M → Remove all asterisks (*) from selected text
  • Ctrl+I → Convert selected text to uppercase
  • Ctrl+* → Insert token wrapped in **
  • / → Focus search
  • Esc (in field) → Cancel

Type : for emoji autocomplete
